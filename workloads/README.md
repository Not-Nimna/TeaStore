# Workloads

This directory is the report-facing home for TeaStore workload definitions.

Canonical runnable JMeter plans remain in `examples/jmeter/`:

- `examples/jmeter/teastore_browse_nogui.jmx`
- `examples/jmeter/teastore_login_nogui.jmx`
- `examples/jmeter/teastore_mixed_checkout_nogui.jmx`

Use this directory in the report to describe the workload set, while the
actual plans stay in `examples/jmeter/`.
