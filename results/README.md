# Results

Recommended layout for generated experiment artifacts:

- `results/raw/` for raw `.jtl` files, time-series CSVs, and Docker stats CSVs
- `results/figures/` for PNG figures used in the report
- `results/tables/` for CSV tables and comparison summaries

This directory is intentionally kept out of source control except for empty
placeholder files so you can store experiment outputs here cleanly.
